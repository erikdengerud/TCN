# Shape dataset

This is  a dataset with 3 shapes, 3 frequencies and 3 noise setting.
We use the dataset to test embeddings and use a 2 dimensional embedding space.

## Data

The dataset contains N time series of length t.

### Shapes:
* Triangular
* Rectangular
* Sine

### Noise:
* No noise
* IID
* Matern

### Frequency:
* 2
* 10
* 20
